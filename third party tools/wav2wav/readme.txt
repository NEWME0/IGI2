Innerloop Studios WAV->WAV converter v1.0.0


Программа предназначена для конвертации звуковых файлов фирмы Innerloop
Studios в стандартные wav файлы. Программа является консольной, поэтому
её лучше всего запускать из FAR'а или Total Commander'а. В качестве
аргумента ей передаётся имя файла для преобразования (обязательный
параметр, может содержать метасимволы * и ?).


Примеры запуска :
1. wav2wav.exe innerloop.wav
2. wav2wav.exe *.wav


Работа программы была проверена на звуковых файлах из игр Project IGI и
Project IGI 2 : Covert Strike.
